# 个人中心页面设计规范补充

## 一、页面布局

### 1.1 整体结构

**标准实现**：`Profile.vue`

**布局结构**：
- 页面标题 + 副标题
- 左右分栏布局（左侧用户信息，右侧详细信息）
- 左侧：头像 + 基本信息 + 信用积分卡片
- 右侧：标签页（基本信息表单 + 账户信息列表）

```vue
<template>
  <div class="profile-container">
    <!-- 页面标题 -->
    <div class="page-header">
      <h1 class="page-title">个人中心</h1>
      <p class="page-subtitle">管理您的个人信息和账户设置</p>
    </div>

    <div class="profile-content">
      <el-row :gutter="24">
        <!-- 左侧：用户信息卡片 -->
        <el-col :xs="24" :sm="24" :md="8" :lg="8">
          <div class="user-info-card">
            <!-- 头像区域 -->
            <!-- 用户基本信息 -->
            <!-- 信用积分卡片 -->
          </div>
        </el-col>

        <!-- 右侧：详细信息 -->
        <el-col :xs="24" :sm="24" :md="16" :lg="16">
          <div class="info-tabs-card">
            <el-tabs v-model="activeTab">
              <!-- 基本信息表单 -->
              <!-- 账户信息列表 -->
            </el-tabs>
          </div>
        </el-col>
      </el-row>
    </div>
  </div>
</template>
```

## 二、左侧用户信息卡片

### 2.1 卡片样式

```css
.user-info-card {
  background-color: #FFFFFF;
  border-radius: 12px;
```

### 2.2 头像区域

**设计要点**：
- 头像尺寸：120px
- 渐变背景：`linear-gradient(135deg, #769fcd 0%, #b9d7ea 100%)`
- 白色边框：4px
- 阴影效果：`0 4px 12px rgba(118, 159, 205, 0.2)`

```css
.user-avatar {
  background: linear-gradient(135deg, #769fcd 0%, #b9d7ea 100%);
  font-size: 48px;
  font-weight: bold;
  color: white;
  border: 4px solid #e8f4f8;
  box-shadow: 0 4px 12px rgba(118, 159, 205, 0.2);

.change-avatar-btn {
```

## 八、与其他页面的一致性

### 8.1 与认证页面的一致性

**相同点**：
- 使用相同的配色方案
- 卡片样式一致（圆角、边框、阴影）
- 按钮样式一致（渐变背景、悬停效果）
- 输入框样式一致

**不同点**：
- 个人中心使用左右分栏（8:16），认证页面使用左右分栏（60:40）
- 个人中心有标签页，认证页面没有
- 个人中心有信息列表，认证页面只有表单

### 8.2 与主布局的一致性

- 使用相同的页面背景色（#f7fbfc）
- 卡片样式保持一致
- 图标颜色保持一致
- 交互效果保持一致
