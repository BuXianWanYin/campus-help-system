## 使用说明

### 登录方式

1. **访问登录页面**
   - 前端地址: `http://localhost:3000/login`
   - 后端 API: `POST /api/auth/login`

2. **登录方式**
   - 密码登录：输入邮箱和密码
   - 验证码登录：输入邮箱和验证码

### 测试场景

#### 管理员测试场景

1. **实名认证审核**
   - 使用 `test@qq.com` 提交实名认证

## 注意事项

⚠️ **重要提示**：

1. **仅用于测试环境**
   - 这些账号仅用于开发和测试环境
   - 生产环境请勿使用这些账号

2. **密码安全**
   - 测试账号密码较为简单，仅用于测试
   - 生产环境必须使用强密码

3. **数据清理**
   - 测试完成后，建议清理测试数据
   - 避免测试数据影响生产环境

4. **账号状态**
   - 测试账号可能被重置或删除
   - 如有需要，请联系管理员创建新的测试账号

## 创建新测试账号

如需创建新的测试账号，可以通过以下方式：

1. **前端注册**
   - 访问注册页面: `http://localhost:3000/register`
   - 填写邮箱和密码进行注册

2. **数据库直接创建**
   - 连接数据库 `campus_help`
   - 在 `user` 表中插入新用户记录
   - 密码需要使用 BCrypt 加密

3. **管理员创建**
   - 管理员可以在管理后台创建用户（如果已实现该功能）

## 密码加密说明

系统使用 BCrypt 对密码进行加密存储。如果需要在数据库中直接创建用户，密码需要先进行 BCrypt 加密。

**BCrypt 加密示例**（Java）：
```java
import org.springframework.security.crypto.bcrypt.BCryptPasswordEncoder;

BCryptPasswordEncoder encoder = new BCryptPasswordEncoder();
String encodedPassword = encoder.encode("原始密码");
```
